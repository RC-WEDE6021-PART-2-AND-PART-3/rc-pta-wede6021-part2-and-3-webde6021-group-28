================================================================================
PASTIMES - IMAGES DIRECTORY
================================================================================

This directory contains all images for the Pastimes clothing marketplace.

REQUIRED DEFAULT IMAGES:
------------------------
1. default-avatar.png     - Default user profile picture (200x200px)
2. default-clothing.jpg   - Default clothing item placeholder (600x800px)
3. hero-bg.jpg           - Homepage hero background image (1920x1080px)
4. logo.png              - Pastimes logo (transparent PNG)

CATEGORY IMAGES:
----------------
- category-tops.jpg
- category-bottoms.jpg
- category-dresses.jpg
- category-outerwear.jpg
- category-footwear.jpg
- category-accessories.jpg
- category-activewear.jpg

CLOTHING ITEM IMAGES:
---------------------
Images uploaded by sellers will be stored here with unique filenames.
Naming convention: {clothingID}_{timestamp}.{extension}

Example: 15_1699012345.jpg

RECOMMENDED IMAGE SPECIFICATIONS:
---------------------------------
- Product images: 600x800px (portrait orientation)
- Category images: 400x400px (square)
- Hero images: 1920x1080px (landscape)
- Profile pictures: 200x200px (square)
- File formats: JPG, PNG, WEBP
- Maximum file size: 5MB

PLACEHOLDER IMAGES:
-------------------
You can download free placeholder images from:
- https://unsplash.com (search for "clothing", "fashion")
- https://pexels.com
- https://picsum.photos (for random placeholders)

IMPORTANT NOTES:
----------------
1. Ensure all images are web-optimized for fast loading
2. Use descriptive filenames for SEO purposes
3. All uploaded images are processed by import_images.php script
4. Back up this directory regularly
