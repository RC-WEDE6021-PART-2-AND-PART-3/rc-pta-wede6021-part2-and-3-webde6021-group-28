# Images Directory

This directory contains all user-uploaded images and assets for the Pastimes application.

## Default Assets
- `default-avatar.png` - Default user profile picture
- `default-clothing.jpg` - Default clothing item image

## Uploaded Items
User-uploaded clothing images are stored here with unique filenames generated at upload time.

## Usage Notes
- All images should be optimized for web (JPG, PNG, WebP)
- Maximum file size: 5MB
- Images are referenced in code with relative path: `images/filename.extension`
